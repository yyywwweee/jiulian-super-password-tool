# macOS 平台

macOS 版本使用 Swift / AppKit 实现，SwiftPM target 源码位于：

```text
platforms/macos/Sources/JiulianSuperPasswordTool/
```

macOS 图标等设计/发布资产统一位于：

```text
Assets/AppIcon/macos/
```

后端 helper 统一放在：

```text
shared/backend/jiulian_backend_helper.py
```

构建脚本：

```bash
./scripts/build_release.sh
```
