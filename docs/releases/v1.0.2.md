## 更新说明

### Windows

- 修复 Windows 版初始窗口高度不足导致运行日志区域被遮挡的问题。
- 修正 Tkinter 主窗口 grid 伸缩权重，让内容区域随窗口大小正确自适应。
- 启动后根据实际内容请求尺寸自动调整窗口，减少用户首次打开后手动拖拉窗口的需要。
- Windows exe 后端已移除对运行环境自带 `telnetlib` 的依赖，改用随项目 vendored CPython 3.12 `telnetlib.py` 兼容层，避免 Python 新版本缺少 `telnetlib` 导致后端无法加载。

### 文档

- 新增项目设计文档，说明整体架构、模块职责、前后端 JSON 协议、核心业务流程、构建发布、安全约束和兼容性风险。
- 补充并修复 Python `telnetlib` 在新版本 Python 中不可用的兼容性风险。

### macOS

- 本版本同步更新版本号和构建号。
- 后端已移除对运行环境自带 `telnetlib` 的依赖，改为随项目 vendored CPython 3.12 `telnetlib.py` 兼容层，降低 Python 版本变化导致的运行风险。

## 注意事项

- 本工具仅用于本人拥有或被授权维护的设备，请勿用于未授权设备。
- Windows 版为未签名 exe，首次运行可能出现系统安全提示。
- macOS 发布包为 ad-hoc 签名，未进行 Apple Developer ID 公证。
