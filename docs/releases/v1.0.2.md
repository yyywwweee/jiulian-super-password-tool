# 九联光猫获取超级密码工具 v1.0.2

## 更新说明

### Windows

- 修复 Windows 版初始窗口高度不足导致运行日志区域被遮挡的问题。
- 修正 Tkinter 主窗口 grid 伸缩权重，让内容区域随窗口大小正确自适应。
- 启动后根据实际内容请求尺寸自动调整窗口，减少用户首次打开后手动拖拉窗口的需要。
- Windows exe 使用 Python 3.12.2 重新打包，避免 Python 3.14 缺少 `telnetlib` 导致后端无法加载。

### 文档

- 新增项目设计文档，说明整体架构、模块职责、前后端 JSON 协议、核心业务流程、构建发布、安全约束和兼容性风险。
- 补充 Python `telnetlib` 在新版本 Python 中不可用的兼容性风险说明。

### macOS

- 本版本同步更新版本号和构建号。
- 当前 macOS 版仍调用用户环境中的 `python3` 运行后端 helper。建议运行环境使用 Python 3.12 或更低版本；如果系统 Python 升级到 3.13+，可能因缺少 `telnetlib` 导致后端失败。

## 注意事项

- 本工具仅用于本人拥有或被授权维护的设备，请勿用于未授权设备。
- Windows 版为未签名 exe，首次运行可能出现系统安全提示。
- macOS 发布包为 ad-hoc 签名，未进行 Apple Developer ID 公证。
