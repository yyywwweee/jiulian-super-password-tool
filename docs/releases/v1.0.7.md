## 更新说明

这是 v1.0.6 之后的修复发布版本，主要修复 macOS 版运行日志框空白问题，并包含近期 XML 保存兼容性修复。

### macOS 日志框修复

- 修复 macOS 版运行日志框可能显示为空的问题。
- 显式初始化 `NSTextView` 的 frame、滚动尺寸和 text container，避免 textStorage 有内容但可视区域为 0。
- 给日志滚动区域增加标准边框，便于确认日志区域位置。
- 保持后端流式 JSON 日志输出方式不变，前端继续实时显示步骤进度和错误信息。

### XML 保存兼容性

- 修复过滤回传保存的 XML 缺少 `<Config Name="ROOT">...</Config>` 单根节点的问题。
- 过滤模式下保存前会自动补全 ROOT 节点，便于浏览器或 XML 工具正常打开。

### 构建与归档

- macOS 与 Windows 继续统一使用 GitHub Actions 云端构建。
- 发布产物继续使用英文文件名，避免 GitHub 附件名截断。
- 发布后会把云端生成的 macOS 与 Windows 产物回拷到本机和 NAS 归档目录。

### 注意事项

- macOS 发布包为 ad-hoc 签名，未进行 Apple Developer ID 公证。
- Windows exe 未进行代码签名时，SmartScreen 可能提示风险，需要用户手动允许运行。
