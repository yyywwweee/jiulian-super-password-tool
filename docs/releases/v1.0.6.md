## 更新说明

这是 v1.0.5 之后的完整发布版本，主要修复 Windows 图标显示质量与 macOS 窗口拉伸布局问题。

### Windows 图标修复

- 修复 Windows 版标题栏图标与任务栏图标显示。
- 调整 `iconbitmap` 与 `WM_SETICON` 设置逻辑，避免 Windows 取到过小图标。
- 重新整理 Windows `.ico` 资产，使用同一 1024 图标母版生成多尺寸图标。
- 当前 `.ico` 包含 9 个尺寸：16/20/24/32/40/48/64/128/256。

### macOS 窗口布局修复

- 修复拖大窗口时界面整体下沉、窗口顶部出现空白的问题。
- 顶部控件现在锁定在窗口顶部区域。
- 运行日志区域会随窗口高度变大，方便查看更多日志。

### 图标资产维护

- 移除未采用的 `AppIcon.svg` 方案。
- 明确 `scripts/generate_icon.py` 仅作为图标资产维护脚本使用。
- 日常构建直接使用仓库中已归档的 `.icns` / `.ico` 设计资产，不在构建时重新生成图标。

### 构建流程

- 移除 macOS CI 中不必要的 Pillow 安装步骤。
- 保持 macOS 与 Windows 发布产物命名一致。
